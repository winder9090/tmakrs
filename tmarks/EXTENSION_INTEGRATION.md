## 技术实现

### 下载功能
```typescript
const handleDownload = () => {
  const link = document.createElement('a')
  link.href = '/tmarks-extension.zip'
  link.download = 'tmarks-extension.zip'
  document.body.appendChild(link)
  link.click()
  document.body.removeChild(link)
}
```

### 路由保护
- 插件页面需要登录才能访问
- 使用 `ProtectedRoute` 组件保护
- 未登录用户会被重定向到登录页

## 注意事项

1. **插件文件更新**
   - 更新插件时，需要同时更新 `public/tmarks-extension.zip`
   - 建议在页面上更新版本号和更新时间

2. **浏览器兼容性**
   - 插件仅支持 Chromium 内核浏览器
   - 不支持 Firefox、Safari 等其他浏览器

3. **文件大小**
   - 当前插件文件约 4.8 MB
   - 建议优化插件体积以加快下载速度

4. **安全性**
   - 插件需要配置 API Key
   - 建议在文档中强调 API Key 的安全性
   - 提醒用户不要分享 API Key
